## 它到底能做什么？

操作逻辑很简单：导入文件 → 调整扫描效果参数 → 导出。

效果参数可以自由调节，包括：

- 黑白 / 彩色模式
- 旋转角度（模拟拍歪了那种感觉）
- 亮度、对比度、分辨率
- 噪点、纸张发黄等做旧效果
- 模拟边框

调完之后，输出的 PDF 看起来就像是用扫描仪扫出来的，文字内容也不那么容易被直接复制。

整个处理过程**全部在本地浏览器完成**，断网状态也能用。你的文件不会经过任何服务器，对隐私敏感的场景很友好。
